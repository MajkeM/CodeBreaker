Place your audio files here.

Recommended formats:
- Web: OGG (good compression + browser support) or MP3.
- Music: use loop-friendly OGG files for background music (bgm_*.ogg).
- SFX: short OGG/MP3 files (sfx_*.ogg).

Example names used in scenes:
- calm_loop.ogg  -> background music
- paper_flip.ogg -> sfx for page flip

Notes:
- Keep filenames lowercase, preferably with underscores.
- If you use Vite dev server, files in this folder are served without extra config at /assets/audio/FILE
- Replace placeholders in `scenes/*.json` with your filenames (just the file name, e.g. "calm_loop.ogg").
